## Image Classifier

This project is an implementation of an image classifier using Convolutional Neural Networks. It classifies to 10 classes with labels mapped as below.

* 'deer' &rarr; 0
* 'horse' &rarr; 1
* 'car' &rarr; 2
* 'truck' &rarr; 3
* 'small mammal' &rarr; 4
* 'flower' &rarr; 5
* 'tree' &rarr; 6
* 'aquatic mammal' &rarr; 7
* 'fish' &rarr; 8
* 'ship' &rarr; 9
